# Política de Protección de Datos Personales

**Objetivo:** Definir el tratamiento jurídico y técnico de la información personal, en cumplimiento con la **Ley Orgánica de Protección de Datos Personales (LOPDP)** de Ecuador.

---

## Principios Rectores

- **Minimización:** Solo se recolectan los datos estrictamente necesarios para crear un perfil laboral (CV).
- **Finalidad:** El uso se limita exclusivamente a la intermediación laboral y validación de identidad.
- **Proporcionalidad:** El tratamiento es adecuado y no excesivo en relación con el servicio gratuito prestado.

---

## Tipología de Datos

- Datos identificativos  
- Datos de contacto  
- Datos profesionales (experiencia, formación, CV)  
- **Prohibición expresa:** No se solicitan datos sensibles (religión, salud, orientación política, etnia, biométricos, etc.)

---

##  Conservación de Datos

- **Cuentas Activas:** Conservación indefinida mientras el usuario utilice la plataforma.  
- **Cuentas Inactivas:** Eliminación automática después de **24 meses** de inactividad, con notificación previa al usuario.  
- **Logs de Seguridad:** Retención mínima de **90 días** para fines de auditoría forense y detección de incidentes.

---

##  Seguridad Base

- Cifrado en tránsito mediante **TLS 1.3**.  
- Hashing de contraseñas con **Argon2**.  
- Controles de acceso basados en roles (**RBAC**) para limitar privilegios.  

---

##  Derechos ARCO+

Se establece un canal oficial para que los usuarios ejerzan sus derechos de:
- **Acceso**  
- **Rectificación**  
- **Eliminación**  
- **Oposición**  
- **Portabilidad**
